## Table of contents

## Cursor motions

### Normal Mode

- h - move to left
- j - move to down
- k - move to up
- l - move to right
- gg - move to the top
- G - move to the end

## Operators

### Normal Mode

- y/d/c - yank/delete/change
- gu/gU - change case
- v - start visual mode per character
- V - start visual mode linewise

- va{% - delete all inside the curly braices included it
